
# ToDo-List Manager
C++ console-based ToDo List Manager that allows user to manage their daily tasks.

## Features

1. Add new tasks
2. View all tasks
3. Mark tasks as completed
4. Delete tasks
5. Save tasks to a file

### Usage

This project was developed as a learning exercise to practice core C++ programming concepts, including structures, functions, arrays, and file handling while building a simple task management system.
